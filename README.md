## Features

- **3D Interactive Map**: Explore Dakar's art spaces through a hand-drawn, stylized terrain
- **54 Cultural Locations**: Galleries, museums, cultural centers, and artist studios
- **Scroll Storytelling**: Immersive place pages with rich content and parallax effects
- **Hand-Drawn Aesthetic**: Inspired by Senegalese art and African design principles

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Three.js + React Three Fiber
- GSAP (animations)
- Tailwind CSS
- Mapbox GL JS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Design System

### Colors

- Deep Indigo: `#1A2B4A` (ocean, depth)
- Cream: `#F5F1E8` (land, backgrounds)
- Terracotta: `#C4502A` (galleries, primary accent)
- Ochre: `#D4A03E` (cultural centers, warmth)
- Sand: `#E8DFD0` (subtle backgrounds)
- Deep Brown: `#2D1F14` (studios, text)

### Typography

- Display: Bebas Neue (headlines, place names)
- Body: Space Grotesk (UI, descriptions)
- Editorial: Cormorant Garamond (quotes, accents)
